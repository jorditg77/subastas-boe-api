# Política de Privacidad y Tratamiento de Datos - Subastas BOE API

## 1. Fuente de los datos

Esta API extrae información del portal oficial de subastas del Boletín Oficial del Estado (`subastas.boe.es`), datos de carácter público sujetos a condiciones de reutilización de la información pública.

## 2. Datos personales

La API no almacena ni transmite datos personales de personas físicas identificables. Implementamos un filtro de redacción automática (PII redaction) que elimina o anonimiza:

- Nombres completos de personas físicas.
- Números de identificación (NIF, DNI, NIE).
- Direcciones exactas de particulares.
- Datos de contacto personales.

## 3. Finalidad

La finalidad de la API es proporcionar información agregada sobre bienes inmuebles y activos en subasta pública para fines de análisis inmobiliario, inteligencia de negocio y alimentación de agentes de IA.

## 4. Responsabilidad

El usuario es responsable del uso que haga de los datos. Esta API no sustituye la consulta de la fuente oficial ni constituye asesoramiento legal, fiscal o inmobiliario. Antes de participar en una subasta, el usuario debe verificar la información directamente en `subastas.boe.es` y mediante nota simple registral.

## 5. Contacto

Para ejercer derechos de acceso, rectificación, supresión u oposición, contactar a través de la ficha de RapidAPI.
