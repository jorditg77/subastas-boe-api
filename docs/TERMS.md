# Términos de Uso - Subastas BOE API

## 1. Aceptación

Al usar esta API, aceptas estos términos. Si no estás de acuerdo, no uses el servicio.

## 2. Uso permitido

- Consulta de datos agregados de subastas públicas.
- Integración en aplicaciones PropTech, CRMs y agentes de IA.
- Análisis de mercado inmobiliario.

## 3. Uso prohibido

- Reidentificación de personas físicas a partir de datos anonimizados.
- Uso de la información para acoso, discriminación o fines ilícitos.
- Sustracción masiva de datos mas allá de los límites del plan contratado.
- Reventa de los datos en bruto sin valor añadido.
- Saturación intencionada de la infraestructura.

## 4. Limitación de responsabilidad

La API se proporciona "tal cual". El operador no garantiza la exactitud, integridad o actualidad de los datos. El usuario debe verificar siempre la información en la fuente oficial antes de tomar decisiones de inversión.

## 5. Cambios

Estos términos pueden actualizarse. El uso continuado del servicio implica la aceptación de los cambios.
